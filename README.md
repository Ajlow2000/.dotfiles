# Ajlow's Dotfiles
Backups of my various config files.  Feel free to use any of my dotfiles below.  
They are organized for easy deployment with the use of a program called [stow](https://www.gnu.org/software/stow/manual/stow.html)


## Programs with corresponding dotfiles:
* Local Scripts (~/.local/bin/)
* Bash
* Vim (Neovim)
* i3-gaps
* Awesome WM
